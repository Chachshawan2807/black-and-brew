    หน้าหลัก

    เห็นทันทีว่าวันนี้และพรุ่งนี้ใครเข้ากะ — อัปเดตแบบเรียลไทม์

  2. 2

    แดชบอร์ดพนักงาน

    ลงเวลา ดูรายชื่อกะ ตารางรายเดือน และสภาพอากาศ

  3. 3

    ตารางงาน

    ลากวางจัดกะ 6:30 / 7:00 / 8:00 ซิงก์วันหยุดราชการ

  4. 4

    บันทึกการซ่อม

    จดประวัติซ่อมเครื่องชง ตู้เย็น และอุปกรณ์ในร้าน

  5. 5

    คลังสินค้า

    แก้ไขสต็อกในตาราง บันทึกอัตโนมัติ แจ้งเตือนเมื่อของใกล้หมด

  6. 6

    ตรวจนับคลัง

    นับสต็อกจริงเทียบกับในระบบ ลดความคลาดเคลื่อน

  7. 7

    วิเคราะห์ตลาด

    ดูเทรนด์เมนูฮิตและคู่แข่งรอบย่านลำลูกกา

  8. 8

    จัดการยอดขาย

    อัปโหลด Excel ดูกราฟหมวดหมู่ รู้ว่าเมนูไหนขายดี

เทคโนโลยีหลัก (Tech Stack)
Next.js 16 React 19 Supabase Tailwind 4 Vercel Gemini AI
  • ฐานข้อมูล PostgreSQL

    Supabase Edge ประเทศไทย — เก็บข้อมูลร้านบนคลาวด์

  • ซิงก์เรียลไทม์

    Supabase Realtime — แก้ไขแล้วอัปเดตทุกเครื่องทันที

  • ความปลอดภัย RLS + PIN

    Row Level Security + ยืนยัน PIN ก่อนแก้ไขข้อมูล

สถาปัตยกรรม (Architecture)
  • Hybrid PPR

    โครงหน้าโหลดเร็ว + ข้อมูลสดจาก Server Actions

  • Server Actions + API

    บันทึกข้อมูลผ่าน Server Actions, AI/LINE ผ่าน API Routes

  • AI ToolLoop + Gateway

    Gemini ดึงข้อมูลร้านผ่าน AI Data Gateway อย่างปลอดภัย

  • PWA + 2 ภาษา

    ติดหน้าจอมือถือได้ (Service Worker) + next-intl ไทย/อังกฤษ

Flow: เบราว์เซอร์ → Next.js App Router → Server Actions / API → Supabase PostgreSQL
